дҪңз”Ё

з»ҷжү§иЎҢиҜҘж–Үд»¶зҡ„з”ЁжҲ·дёҙж—¶иөӢдәҲеҸҰдёҖдёӘз”ЁжҲ·(з»„)зҡ„жқғйҷҗгҖӮ

жҜ”еҰӮи®ҫзҪ®еҜҶз Ғ

/etc/shadowж–Үд»¶еҸӘиғҪиў«rootз”ЁжҲ·дҝ®ж”№гҖӮ
дҪҶжҳҜеҖҹеҠ©passwdе‘Ҫд»Ө, жҷ®йҖҡз”ЁжҲ·еҸҜд»Ҙдёҙж—¶еҸҳжҲҗrootжқҘжү§иЎҢиҝҷдёӘpasswdе‘Ҫд»Ө,
иҝӣиҖҢдҝ®ж”№/etc/shadowж–Үд»¶гҖӮ

еҲҮжҚўз”ЁжҲ· SUID

еҸӘжңүеҸҜд»Ҙжү§иЎҢзҡ„дәҢиҝӣеҲ¶зЁӢеәҸжүҚиғҪи®ҫе®ҡSUIDжқғйҷҗгҖӮ
SUIDеҸҜд»ҘзңӢжҲҗжҳҜSwitch User ID(дёҖдёӘзһҺзҢңпјү, е°ұжҳҜеҲҮжҚўз”ЁжҲ·ID.

  1. еҲӣе»әдёӨдёӘж–Үд»¶file1е’Ңfile2
  2. ж·»еҠ SUIDжқғйҷҗ, жҲ–иҖ…chmod u+s file
  3. еҸ–ж¶ҲSUIDжқғйҷҗ, жҲ–иҖ…chmod u-s file

еҸҜд»ҘзңӢеҲ°xжү§иЎҢжқғйҷҗзҡ„дҪҚзҪ®иў«sжӣҝжҚўдәҶгҖӮ
4755е’Ң4655зҡ„еҢәеҲ«еңЁдәҺж–Үд»¶жң¬иә«жңүжІЎжңүxжү§иЎҢжқғйҷҗгҖӮ
е№¶дё”еҸҜд»Ҙеҫ—зҹҘs=S+x, е№¶дё”жіЁж„Ҹ!!! еӨ§еҶҷSжҳҜжІЎжңүж„Ҹд№үзҡ„гҖӮ
еҸӘжңүе°ҸеҶҷsжүҚиғҪиў«жү§иЎҢ, жүҚиғҪжӯЈзЎ®и®ҫе®ҡSUIDжқғйҷҗгҖӮ

еҲҮжҚўз”ЁжҲ·з»„ SGID

е’ҢSUIDдёҖж ·, еҢәеҲ«еңЁдәҺз”ЁжҲ·дјҡдёҙж—¶иөӢдәҲж–Үд»¶жүҖеұһз”ЁжҲ·з»„groupзҡ„иә«д»Ҫ, иҖҢдёҚжҳҜз”ЁжҲ·userиә«д»ҪгҖӮSwitch Group ID

жҜ”еҰӮ**locateе‘Ҫд»Ө**

/var/lib/mlocate/mlocate.dbж–Үд»¶еҸӘиғҪиў«slocateз»„жҹҘзңӢгҖӮ
дҪҶжҳҜеҖҹеҠ©locateе‘Ҫд»Ө, жҷ®йҖҡз”ЁжҲ·еҸҜд»Ҙдёҙж—¶еҸҳжҲҗslocateжқҘжү§иЎҢиҝҷдёӘlocateе‘Ҫд»Ө,
иҝӣиҖҢжҹҘзңӢ/var/lib/mlocate/mlocate.dbж–Үд»¶гҖӮ

йҷӨдәҶе’ҢSUIDе·®дёҚеӨҡзҡ„еҠҹиғҪеӨ–гҖӮ
SGIDеҜ№зӣ®еҪ•иҝҳжңүеҸҰдёҖдёӘеҠҹиғҪгҖӮ
е°ұжҳҜиҝӣе…ҘиҜҘзӣ®еҪ•еҗҺ, з”ЁжҲ·зҡ„з”ЁжҲ·з»„дјҡеҸҳжҲҗSGIDеҜ№еә”зҡ„з”ЁжҲ·з»„гҖӮ

  1. rootз”ЁжҲ·еҲӣе»әtestзӣ®еҪ•е№¶иөӢдәҲ777жқғйҷҗ
  2. ahaoз”ЁжҲ·еңЁtestзӣ®еҪ•дёӢеҲӣе»әfile1
  3. rootз”ЁжҲ·дёәtestзӣ®еҪ•иөӢдәҲSGIDжқғйҷҗ
  4. ahaoз”ЁжҲ·еңЁиөӢдәҲSGIDзҡ„testзӣ®еҪ•дёӢеҲӣе»әfile2

еҸҜд»ҘзңӢеҲ°иөӢдәҲSGIDеҗҺ, ahaoз”ЁжҲ·еҲӣе»әзҡ„file2зҡ„жүҖеұһз»„жҳҜrootгҖӮ

йҳІжӯўиў«еҲ йҷӨ sticky bit ( SBIT )

йҷӨдәҶuserзҡ„SUIDгҖҒgroupзҡ„SGIDеӨ–, иҝҳжңүotherзҡ„sticky bitгҖӮ

  1. еҸӘеҜ№зӣ®еҪ•жңүж•ҲгҖӮ
  2. ж–Үд»¶жқғйҷҗдёәrwxrwxrwx, д№ҹе°ұжҳҜ777зҡ„жқғйҷҗгҖӮ
    жҜ”еҰӮ/tmpзӣ®еҪ•гҖӮ

йӮЈд№Ҳд»»дҪ•з”ЁжҲ·йғҪиғҪеҜ№зӣ®еҪ•дёӢзҡ„ж–Үд»¶иҝӣиЎҢиҜ»еҶҷжү§иЎҢж“ҚдҪң, иҝҷжҳҜеҫҲдёҚе®үе…Ёзҡ„гҖӮ

д№ҹе°ұжҳҜиҜҙеңЁ777жқғйҷҗзҡ„зӣ®еҪ•дёӢ, Aз”ЁжҲ·еҲӣе»әзҡ„ж–Үд»¶еҸҜиғҪиў«Bз”ЁжҲ·еҲ йҷӨгҖӮ

  3. е…¶д»–з”ЁжҲ·жӢҘжңүеҶҷжқғйҷҗгҖӮ

  3. ahaoз”ЁжҲ·еҲ йҷӨfileж–Үд»¶жҲҗеҠҹ, еӣ дёәahaoз”ЁжҲ·еҜ№testзӣ®еҪ•жңүwеҶҷжқғйҷҗгҖӮ
  4. rootз”ЁжҲ·еҜ№testзӣ®еҪ•иөӢдәҲsticky bitжқғйҷҗгҖӮ
  5. rootз”ЁжҲ·еҲӣе»әfileж–Үд»¶гҖӮ
